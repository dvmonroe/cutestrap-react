# Cutestrap-React

[Cutestrap](https://github.com/tylerchilds/cutestrap) components built with React

## Getting Started

npm:

```sh
npm install cutestrap-react --save
```

Bower:

```sh
bower install classnames --save
```

yarn:

```sh
yarn add cutestrap-react
```
